## 2. 제3자 제공 및 국외 이전

우산은 AI 기능 제공을 위해 다음 외부 서비스에 데이터를 전송할 수 있습니다:

| 수신자 | 이전 항목 | 이전 목적 | 이전 국가 |
|--------|----------|----------|----------|
| Anthropic (Claude API) | 사용자 입력 텍스트 | AI 응답 생성 | 미국 |
| OpenAI (GPT API) | 사용자 입력 텍스트 | AI 응답 생성 (폴백) | 미국 |
| Google (Gemini API) | 사용자 입력 텍스트 | AI 응답 생성 (폴백) | 미국 |
| Supabase | 계정 정보, 동기화 데이터 | 인증, 클라우드 동기화 | 미국 |

**로컬 AI 모드 사용 시 외부 전송 없음.**

## 3. 개인정보의 안전성 확보 조치

- 모든 클라우드 동기화 데이터는 AES-256 암호화
- API 키는 scrypt 기반 키 파생으로 로컬 암호화 저장
- 감사 로그 (append-only) 기록
- 고위험 작업 실행 전 사용자 승인 필수
