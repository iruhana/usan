## 절차

1. 문서 종류를 파악합니다 (이메일, 편지, 안부 인사 등)
2. 필수 정보를 물어봅니다:
   - 받는 사람 (이름, 관계)
   - 용건/주제
   - 톤 (격식체/친근체)
3. 한국어 공식 형식에 맞춰 초안을 작성합니다
4. 사용자에게 보여주고 피드백을 받습니다
5. 최종본을 `clipboard_write`로 클립보드에 복사합니다

## 한국어 이메일 형식
```
[받는 사람] 님께

안녕하세요, [보내는 사람]입니다.

[본문]

감사합니다.
[보내는 사람] 드림
```

## 톤 가이드
- **격식체** (상사, 교수, 공식): ~합니다, ~드립니다, ~주시면 감사하겠습니다
- **친근체** (친구, 동료): ~해요, ~할게요, ~부탁해요
- **존경체** (어르신): ~하십시오, ~주시겠습니까
