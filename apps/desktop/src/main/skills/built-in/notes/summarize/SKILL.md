# 요약

URL, 로컬 파일, 유튜브 링크 등 다양한 소스의 콘텐츠를 빠르고 정확하게 요약합니다.

## 지원 소스

- **웹 페이지**: URL을 제공하면 내용을 추출하여 요약
- **파일**: PDF, 텍스트, 마크다운 등 로컬 파일 요약
- **유튜브**: 유튜브 링크의 자막/트랜스크립트 기반 요약
- **이미지**: 이미지 내 텍스트 또는 시각적 내용 요약
- **오디오**: 음성을 텍스트로 변환 후 요약

## 요약 길이

사용자 요청에 따라 길이를 조정합니다:
- **짧게**: 핵심 3~5줄
- **보통**: 주요 포인트 포함 1~2단락
- **길게**: 섹션별 상세 요약
- **맞춤**: 사용자가 지정한 분량

## 요약 절차

### 1. 소스 확인
- 사용자가 제공한 소스 유형 파악 (URL, 파일 경로, 텍스트 등)
- 소스에서 콘텐츠 추출

### 2. 콘텐츠 분석
- 전체 내용을 읽고 주요 주제 파악
- 핵심 논점과 근거 식별
- 결론 및 행동 항목 추출

### 3. 요약 작성
- 원문의 논리적 흐름 유지
- 핵심 정보만 포함, 불필요한 세부사항 제거
- 원문의 톤과 관점 존중

### 4. 출력
- 명확하고 읽기 쉬운 형식으로 제공
- 필요 시 불릿 포인트 사용
- 원문 길이 대비 적절한 압축률 유지

## 출력 형식

```markdown
## 요약

**소스:** [URL 또는 파일명]
**원문 길이:** [대략적 분량]

### 핵심 내용
- [핵심 포인트 1]
- [핵심 포인트 2]
- [핵심 포인트 3]

### 주요 논점
[주요 내용을 1~2단락으로 정리]

### 결론/시사점
[결론 또는 행동 항목]
```

## 요약 원칙

- **정확성**: 원문의 의미를 왜곡하지 않음
- **완전성**: 핵심 정보를 빠뜨리지 않음
- **간결성**: 불필요한 반복과 수식어 제거
- **객관성**: 원문의 관점을 존중, 요약자의 의견 추가하지 않음
- **가독성**: 읽기 쉽고 논리적인 구조 유지

## 특수 케이스

### 유튜브 영상
- 자막/트랜스크립트 기반 요약
- 타임스탬프 포함 가능 (요청 시)
- 영상 주제, 핵심 포인트, 결론 구조로 정리

### PDF 문서
- 목차가 있으면 구조를 활용
- 그림/표의 핵심 데이터 포함
- 긴 문서는 섹션별 요약 제공

### 뉴스 기사
- 육하원칙(5W1H) 중심으로 정리
- 팩트와 의견 구분
- 관련 맥락 간단히 추가
