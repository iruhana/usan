# 데일리 브리핑

따뜻하고 간결한 일일 메시지를 생성합니다. 최소한의 맥락만으로도 항상 성공합니다.

---

## 출력 규약 (엄수)

출력은 **브리핑 텍스트만** 포함합니다. 서문, 설명, "완료", 파일 경로, 도구 출력, 코드 펜스 없이.

### 1행 형식 (필수)

1행은 시간대에 맞는 인사로 정확히 시작해야 합니다:

```
{시간대 인사} - 오늘은 {YYYY}년 {M}월 {D}일 {요일}입니다. {날씨 문장}.
```

### 인사 선택 (현지 시간)

| 시간 범위 | 인사 |
|----------|------|
| 05:00~11:59 | 좋은 아침이에요 |
| 12:00~16:59 | 좋은 오후예요 |
| 17:00~21:59 | 좋은 저녁이에요 |
| 22:00~04:59 | 안녕히 주무세요 |
| 알 수 없음 | 좋은 아침이에요 (기본값) |

### 날씨 문장 규칙

**날씨를 가져올 수 있는 경우:**
```
{상태} 하늘, 현재 {온도}도{시간절}{최저절}{강수절}.
```

- 오후 최고 기온이 있으면 시간절: " 오후 기준"
- 그 외 현재 기온 시간절: " 현재"
- 최저 기온이 있으면: `, 최저 {최저}도`
- 강수 확률 30% 이상이면: `, 강수 확률 {확률}%`

**날씨를 가져올 수 없는 경우:**
```
지금은 날씨 정보를 가져올 수 없어요.
```

### 레이아웃 규칙

```
{1행: 날씨 포함 인사}

{생일 섹션 - 있는 경우만}

{일정 섹션 - 있는 경우만}

{리마인더 섹션 - 있는 경우만}

{중요 이메일 섹션 - 활성화되고 있는 경우만}

{앵커 - 실제 우선순위가 있는 경우만}

{마무리 한 줄 - 항상 필수}
```

- 1행 다음 항상 빈 줄
- 각 섹션은 빈 줄로 구분
- 목표 **약 5~15줄**

---

## 톤 앤 무드

- **하루를 위한 부드러운 선물**: 따뜻하고, 차분하고, 공감적이고, 조용히 희망적.
- **잔소리 없음, 긴급함 없음, 생산성 압박 없음.**
- 짧은 줄, 여유 있는 간격, 한눈에 훑기 쉽게.

---

## 실행 절차

### 단계 1: 모드 확인 (대화형 vs 자동화)

**대화형이고 핵심 정보 (위치/시간대/단위) 누락 시:**
- 브리핑 생성 전 누락 정보를 간단히 물어봅니다.

**자동화(크론) 모드:**
- 질문하지 않음. 기본값 사용.
- 파일 생성/수정하지 않음.
- 위치를 알 수 없으면 날씨 생략.

### 단계 2: 날씨 가져오기

날씨 API를 통해 현재 날씨 정보를 수집합니다:
- 위치 설정을 사용
- 설정이 없으면 날씨 생략
- 실패 시 1회 재시도 후 폴백 문장 사용

**파싱 데이터:**
- 상태: 맑음, 흐림, 비 등
- 현재 기온
- 최고/최저 기온
- 강수 확률 (최대값)

**단위:** 사용자 설정에 따라 섭씨(C) 또는 화씨(F). 기본값: 섭씨.

### 단계 3: 데이터 수집

다음 정보를 가용한 소스에서 수집합니다:
- 일정(캘린더) 이벤트
- 리마인더/할 일
- 생일
- 이메일 (활성화된 경우)
- 연락처

### 단계 4: 중요 이메일 분류 (활성화된 경우만)

**중요 이메일 기준 (하나라도 해당되면):**
- 연락처 목록에 있는 사람이 보낸 메일
- 주문 배송 알림
- 결제/거래 확인서
- 환불 관련 메시지
- 구독 갱신 알림
- 은행 알림
- 캘린더 초대
- 계정 보안 이메일

**제외 대상 (다른 기준 해당돼도 제외):**
- 프로모션/마케팅 이메일
- 스팸성 채용 메일
- 제품 업데이트/공지 이메일
- 벤더 뉴스레터

### 단계 5: 브리핑 생성

수집된 모든 데이터를 사용하여 출력 규약에 따라 브리핑을 작성합니다.

**섹션 형식:**

**생일:**
```
🎂 **생일:**
• 오늘: 이름
• 3/20: 이름
```

**일정:**
```
📅 **오늘 일정:**
• 종일: 이벤트 제목
• 오전 9:00: 이벤트 제목
```

**리마인더:**
```
✅ **리마인더:**
• 처방전 받기
```

**중요 이메일:**
```
📧 **확인할 이메일:**
• 발신자: 제목
```

**마무리 한 줄:**
- 필수. 따뜻하고 격려적인 한 마디.

### 단계 6: 출력

브리핑 텍스트만 반환합니다. 다른 것은 출력하지 않습니다.

---

## 예시 출력

```
좋은 아침이에요 - 오늘은 2026년 3월 18일 수요일입니다. 맑은 하늘, 현재 12도, 최저 5도.

🎂 **생일:**
• 오늘: 김민수
• 3/20: 이서연

📅 **오늘 일정:**
• 종일: 건강검진
• 오전 9:00: 팀 스탠드업

✅ **리마인더:**
• 처방전 받기

오늘도 한 걸음씩, 잘 해낼 거예요.
```

---

## 기본값

- **시간대:** 사용자 로컬 시간대. 알 수 없으면 UTC.
- **위치:** 사용자 위치가 있으면 사용. 없으면 자동화 모드에서 날씨 생략.
- **단위:** 사용자 선호 단위. 모르면 섭씨(C).
