## 절차

1. 사용자의 자연어 시간 표현을 파악합니다:
   - "30분 뒤" → delay_minutes: 30
   - "1시간 후" → delay_minutes: 60
   - "오후 3시" → 현재 시간과의 차이를 분으로 계산
   - "내일 아침 8시" → 현재 시간과의 차이를 분으로 계산
2. `set_reminder`로 알림을 설정합니다
3. 확인 메시지를 자연스럽게 전합니다:
   - "네, 30분 뒤에 알려드릴게요"
   - "약 드실 시간에 알림이 울릴 거예요"
4. 알림 목록 요청 시 `list_reminders`로 보여줍니다
5. 취소 요청 시 `cancel_reminder`로 삭제합니다

## 메시지 스타일
- 딱딱한 "리마인더:" 금지
- 자연스럽고 따뜻하게: "약 드실 시간이에요~", "병원 갈 준비 하셔야 해요"
- 최대 48시간 제한 → 초과 시 "48시간까지만 설정할 수 있어요. 더 먼 일정은 메모에 남겨드릴까요?"
