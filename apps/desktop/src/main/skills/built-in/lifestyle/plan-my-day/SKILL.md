# 하루 계획 세우기

에너지 수준에 최적화된 시간 블록 기반 일일 계획을 생성합니다.

## 계획 원칙

1. **아침 프라이밍** — 첫 한 시간은 기상 루틴용, 업무가 아닌
2. **에너지 기반 스케줄링** — 과제 난이도를 에너지 수준에 매칭
3. **시간 블로킹** — 특정 시간대에 특정 작업 배정
4. **상위 3가지 집중** — 오늘 가장 중요한 3가지 성과 식별

## 에너지 윈도우 (개인 리듬에 맞춰 커스터마이징)

- **최대 성과 시간:** 오전 (딥워크, 가장 어려운 작업)
- **보조 피크:** 오후 (집중 작업, 미팅)
- **회복 블록:** 운동, 식사, 휴식을 위한 쉬는 시간
- **마무리:** 저녁 (가벼운 작업, 내일 계획)

## 프로세스

### 1. 맥락 파악
- 기존 일일 노트 확인
- 현재 우선순위 검토
- 고정 약속 확인 (미팅, 통화)
- 어제 미완료 작업 확인

### 2. 상위 3가지 우선순위 식별
- 오늘 반드시 해야 하는 것은?
- 가장 큰 영향을 미치는 것은?
- 마감이 있는 것은?

### 3. 시간 블록 스케줄 작성
- 우선순위를 최대 에너지 시간대에 배정
- 고정 약속 블록 설정
- 블록 사이 버퍼 시간 추가
- 휴식과 회복 포함

### 4. 제약 조건 적용
- 기존 약속 존중
- 개인 시간 보호
- 식사 시간 포함
- 과도한 스케줄링 금지

## 의사결정 프레임워크

작업 전에 항상 묻기:
1. 이것이 상위 3가지 우선순위 중 하나인가?
2. 이것이 오늘의 목표를 향해 나아가게 하는가?
3. 내일로 미룰 수 있는가?

**셋 다 아니오이면 → 하지 마세요**

## 팁

- 시간의 100%를 채우지 마세요 — 20% 버퍼 남기기
- 가장 어려운 작업을 최대 에너지 시간대에 배치
- 비슷한 작업 묶기 (배칭)
- 휴식을 스케줄에 넣기, 기대하지만 말고
- 필요하면 하루 중간에 조정하기
