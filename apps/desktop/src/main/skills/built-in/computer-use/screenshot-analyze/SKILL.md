## 절차

1. `screenshot`으로 현재 화면을 캡처합니다
2. 캡처된 이미지를 분석합니다:
   - 오류 메시지가 있는지 확인
   - 현재 열린 프로그램 확인
   - 화면의 주요 내용 파악
3. 분석 결과를 쉬운 한국어로 설명합니다
4. 오류가 발견되면 해결 방법을 제안합니다

## 주의사항
- 개인정보(비밀번호, 주민번호 등)가 보이면 언급하지 않습니다
- 오류 메시지는 영어여도 한국어로 해석해서 설명합니다
