## 절차

1. 읽을 텍스트를 확인합니다:
   - 사용자가 텍스트를 직접 제공한 경우: 그대로 사용
   - "클립보드 읽어줘": `clipboard_read`로 텍스트 확보
   - 파일 읽어달라는 경우: `read_file`로 파일 내용 확보
2. `speak_text`로 음성을 생성합니다:
   - 한국어 텍스트: 자동으로 한국어 음성 선택
   - 영어/일본어: 자동 감지하여 해당 언어 음성 선택
3. 생성된 오디오 파일 경로를 알려줍니다
4. 긴 텍스트는 적당한 길이로 나누어 읽습니다

## 참고
- 기본 속도는 느리게 (-20%) 설정되어 있습니다
- 속도 조절 요청 시: "빠르게"→+0%, "더 빠르게"→+20%, "느리게"→-30%
