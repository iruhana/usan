### [Chat Layout Over-Segmentation]
**Date:** 2026-03-18
**Symptom:** 홈 화면에 대화 세션 전용 패널이 따로 고정되고, 메모를 포함한 다른 화면도 카드와 섹션이 과하게 나뉘어 데스크톱 앱보다 웹 페이지처럼 보였다.
**Cause:** 대화 전 상태와 대화 중 상태를 같은 2패널 구조로 처리했고, 공용 카드 스타일의 그림자와 경계가 강해 모든 화면에서 분리감이 과도하게 드러났다.
**Resolution:** 대화 전에는 본문 안에 이전 대화 목록만 인라인으로 보여주고, 대화 중에는 상단 드롭다운으로 세션 전환 구조를 바꿨다. 공용 카드 스타일을 더 약하게 조정하고 메모 화면을 좌우 2패널 중심으로 평평하게 재구성했다.
